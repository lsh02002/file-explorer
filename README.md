# 주의 사항!!!

이 소프트웨어는 실제 컴퓨터의 폴더와 파일들을 다루는 기능이 대부분으로 나중에 이 소프트웨어를 사용하거나 소스코드를 수정 후 사용할때 실수로 인해 불안정하면 시스템의 폴더나 파일들이 삭제가 되는등의 상황을 주의 하셔야 할것 같습니다. 그래서 삭제 기능은 휴지통에 버리는 기능만 구현했습니다. 만약을 위해 아무것도 설치가 안되있는 노트북등에 설치하고 수정하는 등 주의가 필요할 것 같습니다! 이 타우리 앱에서 파일 이름을 수정해도 다른 윈도우 시스템에서 UNDO할 수 없습니다. 임의의 수정등으로 인한 불이익은 제가 책임지지 않습니다! 될수 있으면 소스만 보세요.
